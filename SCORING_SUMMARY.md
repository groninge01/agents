# 市场评分系统 - 快速参考

## 🎯 评分标准（0-10分）

### ① 流动性（0-3分）
- < $100k → 0分
- $100k - $300k → 1分  
- $300k - $1M → 2分
- ≥ $1M → 3分

### ② 活跃度（0-2分）
- 最近5分钟：几十笔成交 → 2分
- 最近5分钟：偶尔成交 → 1分
- 最近5分钟：不动 → 0分

### ③ 波动空间（0-2分）
- 历史/日内：≥ 15c → 2分
- 历史/日内：8-15c → 1分
- 历史/日内：< 8c → 0分

### ④ 事件时间结构（0-2分）
- 明确节点（CPI/选举）→ 2分
- 持续发酵 → 1分
- 没节奏 → 0分

### ⑤ 情绪参与度（0-1分）
- 社媒/新闻热 → 1分
- 冷清 → 0分

## ✅ 总分解读

- **≥ 7分**: 可交易 ✅
- **5-6分**: 小仓/观察 ⚠️
- **< 5分**: 跳过 ❌
